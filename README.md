# Why does this measurement matter? 
These measurements Fx, Fy, Fz (forces) and Mx, My, Mz (moments) for all four wheels provides us necessary data for building a Torque vectoring model. My work includes a PDF named "Multi-body simulation", which basically resulted in a system of linear equations, where solving it results in values of Fx, Fy & Fz which is what exactly we need as Input for Torque vectoring model.
# Wheel Force Transducer
As definition goes: The primary device used to measure forces on a wheel is a Wheel Force Transducer, a multiaxial sensor system that replaces the wheel and measures Fx, Fy, Fz (forces) and Mx, My, Mz (moments).
These transducers cost minimum 250K dollar with industry standard but with different approach of Simulation and other methods such as implementing AI can reduce the cost upto 66%.
# Literature Reference:
I give reference which is "The Multibody Systems Approach to Vehicle Dynamics" by Mike Blundell & Damian Harty which is really good for beginners into vehicle dynamics domain but as prerequisite bring good knowledge in Math such as Solving linear equation, matrix & vectors.
